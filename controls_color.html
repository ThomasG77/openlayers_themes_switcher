Map Controls Example

Attach zooming, panning, layer switcher, overview map, and permalink map controls to an OpenLayers window.

Permalink

This example code for the layers switcher is available at http://github.com/ThomasG77/openlayers_themes_switcher
It's only javascript code, we don't rely on a javascript frameworks like ExtJS, JQuery, Motools, Prototype, ...
This layer switcher works at DOM level. We don't play with functions or prototypes.

The colors buttons are derivated from Development Seed one.
We generated them from main html colors but you can generate them for any color.
See http://github.com/ThomasG77/openlayers_themes for further informations.

The fonts color automatically adapts to the opposite color buttons.
It's not always beautifull but you can always see text (Don't suffer from black text on black background !!) You can look on http://gmazzocato.altervista.org/colorwheel/wheel.php to find idea to get good fonts color with color buttons.
